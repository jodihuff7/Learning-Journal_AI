    - AI Models are not deterministic

    - Models learn how to make the prediction by processing an enormous amount of text pulled from the internet
      - Training is like tuning the dials on a big machine
        - Many different continuous values - parameters or weights
          - Changing the parameters will change the probability for any given input
          - Hundreds of billions of parameters
        - Begin at random - repeatedly refined based on example pieces of text
          - Parse all but the last word of the model - more likely to choose the true last word
        - The model starts to get more accurate predictions on existing data and more precise on new data
